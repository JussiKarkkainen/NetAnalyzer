# A Simple tool for analyzing network packets as well as performing ARP spoofing.

## Usage
```
git clone https://github.com/JussiKarkkainen/NetAnalyzer.git
cd NetAnalyzer
```
For passively sniffing network packets:
```
./run.sh -s
```
